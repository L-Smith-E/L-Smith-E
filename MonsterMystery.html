Monster Mystery

Steam Store Page

Monster Mystery is a social seduction game similar to games such as Among Us or Mafia the game involves explorers who must power a generator by sourcing power scattered across the map. However one or two amongst them are actually vampires working to thwart their plans.

This was a project I aided mid development with multiple facets such as rebuilding and refactoring network code including but not limited to: Server travel, Lobby Matchmaking, End Game Scenarios and User Interface in C++. This meant going through a pre established code base and extending functionality without breaking critical features. I was also involved in implementing audio features that were developed by audio engineers who were previously involved with the project. I had a hand in designing and architecting multiple gameplay functions such as player death handling and player onboarding. Additionally I lead porting the game over from Unreal 4.27 to Unreal 5 which included updating level assets to use world partitioning, and various UI fixes.
